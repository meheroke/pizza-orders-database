# Pizza Orders Database Management  
This is a Java and SQL based project for managing a restaurants customers, orders, pizza's, toppings, and inventory using databases.  

Order Management: Create, view, complete orders; track pizzas, apply discounts, modify status of order  
Customer Management: Add and view customer information  
Inventory Management: Track toppings, restock toppings, reduce inventory based on orders placed  
Reporting: generate Reports for Topping Popularity, Profit by Pizza, and Profit by Order Type  
  
  
Language: Java  
Database: SQL database
